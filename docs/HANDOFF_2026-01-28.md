## Current Project State

### Working Server

```bash
npm run dev        # Start without rebuild
npm run start      # Build TypeScript + start
npm run build      # Compile TypeScript only
npm run typecheck  # Type check without compile
```

**Server URL:** http://localhost:8090  
**Health Check:** `curl http://localhost:8090/health`

### API Endpoints (All Functional)

| Method | Endpoint                 | Purpose                                   |
| ------ | ------------------------ | ----------------------------------------- |
| GET    | `/health`                | Health check                              |
| GET    | `/api/v1/creators`       | List/search creators                      |
| GET    | `/api/v1/creators/:id`   | Get single creator                        |
| POST   | `/api/v1/creators`       | Add creator (Zod validated)               |
| POST   | `/api/v1/creators/batch` | Bulk import                               |
| POST   | `/api/v1/match`          | Match creators to brief (uses scoring.ts) |
| POST   | `/api/v1/categorize`     | Auto-categorize (placeholder for LLM)     |
| GET    | `/api/v1/stats`          | Database statistics                       |
